## Summary

A single-page notional, overhead-view map reconstructing four sub-incidents of the October 2023 Western U.S. Event. A disclaimer states all imagery was "artificially generated using https://genai.mil/," that phenomena are enlarged and not to scale, and that all locations are notional.

## Visual description (viewed)

A grayscale aerial/relief map with four labeled blue "observer position" markers, each tied to a sub-incident:

| Marker | Sub-incident | Description on map |
| ------ | ------------ | ------------------ |
| Incident 1 | "Orbs Launching Orbs" | orange "mother orb" launching smaller red "orbs" (rendered upper right as orange dots trailing red) |
| Incident 2 | "Fiery Orb" | a large "fiery orb" projected against a ridgeline at ~1,000 yards (rendered as a sun-like fiery sphere) |
| Incident 3 | "Dark Kite" | a "thin or dark kite-shaped object" with one red and one white light at close range (rendered near a vehicle, lower left) |
| Incident 4 | "Translucent Kite" | a "translucent kite-shaped object" at close range (rendered bottom) |

This confirms the event comprised four distinct reported phenomena, not just the headline "orbs launching orbs."
