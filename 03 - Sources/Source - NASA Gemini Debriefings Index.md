> [!NOTE]
> Tranche 3 added eight large scanned NASA astronaut debriefing transcripts (NASA-UAP-D015 through D022), plus a 1998 Congressional file ([[Source - USG-UAP-D001 Congress White House UFO Correspondence 1998]]). These are indexed together because, on reading, ==the famous astronaut "UFO" sightings are essentially absent from the actual debrief transcripts==. Every anomalous-sounding mention resolves to instrument lights, the booster's flashing rendezvous beacon, airglow/zodiacal light, or fuel-particle "fireflies."

## Analytical note

This index is itself an analytic finding: the primary NASA debriefings do not contain genuine UAP reports. The canonical "astronaut UFO" legends appear only in the 1998 correspondence file ([[Source - USG-UAP-D001 Congress White House UFO Correspondence 1998]]), and there only as quoted-then-debunked secondary material. This complements the vault's existing [[Thesis - Apollo Debris Misidentification Pattern]].
